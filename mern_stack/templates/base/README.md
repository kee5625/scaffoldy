# MERN Base Template

A minimal MERN (MongoDB, Express, React, Node.js) starter generated by `create-mern-app`.

## Tech Stack

- **Frontend:** React + Vite
- **Backend:** Node.js + Express
- **Database:** MongoDB + Mongoose

## Prerequisites

- Node.js `>=18`
- npm
- MongoDB (local instance or MongoDB Atlas)

## Getting Started

### 1) Install dependencies

```/dev/null/bash.txt#L1-3
cd server
npm install
cd ../client && npm install
```

### 2) Configure environment variables

Copy the example file and update values:

```/dev/null/bash.txt#L1-2
cd server
cp .env.example .env
```

If `cp` is not available on your shell, create `server/.env` manually with:

```/dev/null/env.txt#L1-2
PORT=8000
MONGO_URI=mongodb://localhost:27017/mern_base
```

### 3) Run backend

```/dev/null/bash.txt#L1-3
cd server
npm run dev
# or: npm start
```

Backend runs at `http://localhost:8000`.

### 4) Run frontend

Open a second terminal:

```/dev/null/bash.txt#L1-2
cd client
npm run dev
```

Frontend runs at `http://localhost:5173`.

## Available Scripts

### Server (`server/package.json`)

- `npm start` — Start server normally
- `npm run dev` — Start server with Node watch mode

### Client (`client/package.json`)

- `npm run dev` — Start Vite dev server
- `npm run build` — Build production assets
- `npm run preview` — Preview production build
- `npm run lint` — Run ESLint

## Notes

- CORS is configured in `server/server.js` for:
  - `http://localhost:5173`
  - `http://127.0.0.1:5173`
- Update allowed origins before deploying.
